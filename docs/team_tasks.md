# Team Tasks

هذا التقسيم مبدئي ويمكن تعديله أثناء العمل حسب الحاجة.

## غنى النعيمي

المهام المبدئية:

- اختيار Dataset والتحقق منها
- إعداد هيكل المشروع
- كتابة التوثيق الأولي
- Data Preprocessing
- Indexing
- Inverted Index
- تجهيز الأساس الذي ستكمل عليه بقية الخدمات

## شام مخول

المهام المبدئية:

- TF-IDF Representation
- BM25 Representation
- TF-IDF Retrieval
- BM25 Retrieval
- Ranking

## غزل محمد

المهام المبدئية:

- Embedding Representation
- Embedding Retrieval
- Hybrid Retrieval Serial
- Hybrid Retrieval Parallel
- Fusion Method مثل RRF

## شام أبوشاش

المهام المبدئية:

- Query Refinement
- Evaluation
- Charts
- User Interface
- المساعدة في التقرير النهائي

## ملاحظة

هذا التقسيم ليس نهائياً، ويمكن تعديله لاحقاً حسب تقدم العمل وحسب ما يتم إنجازه فعلياً.
