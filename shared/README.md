
## shared/config.py

**نوع:** تعديل على ملف موجود

**الموقع:**
```
shared/config.py
```

**ما الذي تغيّر:**
أضفنا متغيرين جديدين:
- `LARGE_SIZE = 50_000` — حجم العينة الكبيرة
- `TFIDF_LARGE_DIR` — مسار artifacts الـ TF-IDF الكبيرة
- `BM25_LARGE_DIR` — مسار artifacts الـ BM25 الكبيرة

**لماذا:**
لأننا قررنا بناء فهارس على 50,000 وثيقة بدلاً من 10,000 لتحسين نتائج الـ Evaluation.
